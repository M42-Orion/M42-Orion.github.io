类型

字段名 描述 例子
models.CharField 字符串类型 ‘text’
models.TextField 文本类型 ‘tex…xt’
models.EmailField 邮箱类型 ‘123@qq.com’
models.URLField 网址类型 ‘https://www.baidu.com'
models.BooleanField 布尔类型 True False
models.NullBooleanField 可为空的布尔类型 None True False
models.IntegerField 整型 (-2147483648,2147483648)
models.SmallIntegerField 短整型 (-32768,32767)
models.BigIntegerField 长整型 (-9223372036854775808,9223372036854775808)
models.PositiveIntegerField 正整型 (0,2147483647)
models.PositiveSmallIntegerField 短正整型 (0,32767)
models.FloatField 浮点类型 3.14
models.DecimalField 十进制小数 123.321
models.DateField 日期类型 xxxx-xx-xx
models.DateTimeField 日期类型 xxxx-xx-xx xx:xx:xx
models.TimeField 时间类型 xx:xx:xx(时分秒)
models.ImageField 图片类型 xxx.jpg
models.FieldFile 文件类型 任意文件类型

属性

特殊属性

属性名 描述 例子 作用于
max_digits 数字中允许最大位数 10 DecimalField
decimal_places 存储的十进制位数 2 DecimalField
width_field 图片宽 1024 ImageField
height_field 图片高 576 ImageField
upload_to 保存上传文件的本地文件路径，该路径由MEDIA_ROOT设置 ‘xx/xxx/xx.xx’ ImageField,FieldFile

公共属性

属性名 描述 例子 作用于
null 值是否为设为空 True False
blank 值是否为设为空 True False
primary_key 设置主键 True 整型
auto_now 时间自动添加 True 时间类型
auto_now_add 自动添加时间，但仅在创建时间的时候 True 时间类型
max_length 字段长度 字符串类型
default 默认值 xxx
verbose_name admin中显示的名字 name
db_columu 数据库字段名 age
unique 唯一索引 True
db_index 普通索引 True

表关联方法

字段名 描述 例子
ForeignKey 一对多
OneToOneField 一对一
ManyToManyField 多对多
属性名 描述 例子
related_name 关联表的名 related_name=’profile’
on_delete 外键的删除对策 on_delete=models.SET_NULL(CASCADE,PROTECT)
