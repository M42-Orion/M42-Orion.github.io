Django第一弹

Django的名字由来

Django 命名由来官方网站只给出了一句话：
To this day, he's considered one of the best guitarists of all time, Listen to his music. You'll like it.
Django 是以一个名叫 Django Reinhardt 吉他手的的名字来命名的。因为程序员的世界不能缺少音乐，开发者在他的音乐中得到灵感，为了感谢纪念这位吉他手，所以 Django 的名字也由此诞生！
Django 的发音也非常有趣，大写字母 D 不发音，正确的发音是”栈go”,而很多程序员则会把它读成是”D将go”。

文件介绍

HelloWorld: 项目文件夹。
manage.py: 命令工具文件，用于与项目交互，如启动项目。
HelloWorld/__init__.py: 一个空文件，告诉 Python 该目录是一个 Python 包。
HelloWorld/asgi.py: 部署的时候用到，暂时不管。
HelloWorld/settings.py: 配置文件。
HelloWorld/urls.py: 项目路径配置。
HelloWorld/wsgi.py: 部署的时候用到，暂时不管。

admin.py:管理员相关。
apps.py：暂时不管
migrations/__init__.py：这个文件夹下都是自动生成的，不管
models.py：数据模型。
tests.py：用于测试。
views.py：视图部分，MVC中的V。

settings 变量介绍

BASE_DIR：根目录。
SECRET_KEY :一个与安全有关的东西 。不管。
DEBUG：用于开发与上线的调试。
ALLOWED_HOSTS：包含你允许访问的域名，若是同意所有，则’*’。
INSTALLED_APPS：一些应用，包括more应用，如管理员、用户、会话、消息等。
MIDDLEWARE：中间件。
TEMPLATES：与模板有关，比如模板引擎。
WSGI_APPLICATION：不管。
DATABASES：数据库。
AUTH_PASSWORD_VALIDATORS：用户密码相关。
LANGUAGE_CODE：语言配置，中文请用：’zh-hans’
TIME_ZONE = 时间配置，中国请用：’Asia/Shanghai’
STATIC_URL = 静态文件
